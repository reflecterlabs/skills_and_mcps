# Daydreams Skills

Curated skills from the Daydreams ecosystem for building AI agents and deploying to production.

## Skills

- **lucid-agent-sdk** - Build AI agents with the Lucid Agents SDK
- **lucid-client-api** - Interact with the Lucid Client API
- **api-research** - Research and discover APIs for agent development
- **railway-deploy** - Deploy agents to Railway infrastructure

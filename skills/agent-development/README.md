# Agent Development Skills

This directory contains skills for building and working with AI agents, specifically the Lucid Agents ecosystem.

## Skills

- **lucid-agent-sdk**: Skill for working with the Lucid Agents SDK - a TypeScript framework for building and monetizing AI agents
- **lucid-client-api**: Skill for interacting with the Lucid Client API - a multi-agent runtime system

## Usage

Each skill contains a SKILL.md file with detailed instructions and examples.

# Starknet Development MCPs

This directory contains Model Context Protocols for Starknet blockchain development, including Cairo code assistance, smart contract libraries, and gaming frameworks.

## MCPs

- **cairo-coder**: MCP for Cairo code generation and assistance
- **openzeppelin**: MCP for OpenZeppelin Cairo contracts
- **dojo**: MCP for Dojo gaming framework on Starknet

## Usage

Each MCP should have its own subdirectory with configuration and documentation.
